# Quality Gate

Use this skill before a commit or release to verify scope, tests, docs, security, dependencies, and risks.

Follow `AGENTS.md` and use the relevant role profile from `docs/agent-profiles.md`.

Always output:
1. Findings or changes.
2. Validation.
3. Risks.
4. Suggested commit message.
