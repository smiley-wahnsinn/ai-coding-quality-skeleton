# Lazy Quality Task

Use this skill for any implementation task that must remain small, safe, and reviewable.

Follow `AGENTS.md` and use the relevant role profile from `docs/agent-profiles.md`.

Always output:
1. Findings or changes.
2. Validation.
3. Risks.
4. Suggested commit message.
