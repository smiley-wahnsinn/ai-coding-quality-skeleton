# Prompt Guidelines

## Prompt contract

```text
Goal:
Context:
Allowed scope:
Forbidden scope:
Rules:
Acceptance criteria:
Validation:
Output format:
```

## Bad prompt

> Make the app better.

## Good prompt

> Fix the validation bug in one module. Do not touch UI, dependencies, or unrelated files. Add the narrowest useful test and summarize validation.

## Rule

If information does not help the task, remove it.
